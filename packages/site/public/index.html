Trust the files you ship and receive

ProvenancePass creates verifiable, cryptographic receipts for your build & content pipelines, with optional C2PA embedding.

Open Viewer GitHub Docs

How it works

  1. pp keygen — create a signing key
  2. pp wrap --run "your command" — produce a signed receipt
  3. pp embed — (optional) place it into C2PA
  4. pp verify — verify via sidecar or embedded
